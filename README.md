# Dino Evolution — نسخه Flutter/Flame

این پروژه، معادل کامل بازی پایتونی شماست که با Flutter و افزونه Flame بازنویسی شده.
منطق بازی (فیزیک پرش، برخورد، امتیاز، شب/روز شدن، اسکین‌ها) دقیقاً همون چیزیه که
توی فایل پایتون بود.

## قبل از هر کاری
باید Flutter SDK رو روی سیستمتون نصب کرده باشید (از سایت رسمی flutter.dev).
دستور `flutter doctor` رو بزنید و مطمئن بشید مشکلی نشون نمیده.

## مراحل اجرا

۱. وارد پوشه پروژه بشید:
```
cd dino_evolution
```

۲. پکیج‌ها رو نصب کنید:
```
flutter pub get
```

۳. برای تست روی گوشی/شبیه‌ساز وصل‌شده:
```
flutter run
```

## ساخت APK اندروید
```
flutter build apk --release
```
فایل خروجی اینجا قرار می‌گیره:
`build/app/outputs/flutter-apk/app-release.apk`

## ساخت نسخه ویندوز (exe)
```
flutter config --enable-windows-desktop
flutter build windows
```
فایل exe و کنارش فایل‌های لازم، اینجا قرار می‌گیرن:
`build/windows/x64/runner/Release/`

## نکات مهم
- کنترل بازی: لمس هر جای صفحه = پرش (دوبار لمس پشت سر هم = پرش دوم/دبل جامپ).
  برای خم شدن، یک دکمه دایره‌ای پایین-چپ صفحه گذاشته شده که باید نگه دارید.
- ذخیره امتیاز و اسکین انتخابی، به‌جای فایل json (که روی پایتون بود)، داخل
  حافظه اپلیکیشن ذخیره میشه (چون روی اندروید دسترسی مستقیم به فایل کنار برنامه
  ساده نیست). این یعنی امتیازتون هم روی گوشی هم روی ویندوز ذخیره می‌مونه، فقط
  جای فایلش فرق می‌کنه.
- گرافیک پیکسلی هر بار مستقیماً با کد رسم می‌شه (دقیقاً مثل خود پایتون)، نه با
  عکس — پس شکل و رنگ دایناسور/کاکتوس/پرنده همونیه که داشتید.

## اگه موقع build خطا گرفتید
چون این کد بدون اجرای واقعی Flutter نوشته شده (امکان اجرای Flutter روی این
سیستم نبود)، ممکنه بسته به نسخه دقیق Flutter/Flame روی سیستم شما، یکی-دو تا
خط نیاز به اصلاح جزئی داشته باشه (مثلاً تغییر جزئی در نام یک متد). اگه خطایی
دیدید، متن دقیق خطا رو برام بفرستید تا اصلاحش کنم.
